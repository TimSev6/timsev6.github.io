TScopy - Protéger vos images

Nous savons tous à quel point il est difficile de faire de belles images pour son blog ou son site web. Il serait vraiment dommage de se faire voler tout cet effort. Mais heureusement, il existe une solution !

Il suffit d'intégrer ce petit bout de code magique dans votre site et... c'est tout ! Ce script désactivera le clique droit sur toutes les balises IMG de la page, de plus si l'utilisateur tente de télécharger une image, il recevra une alerte lui disant qui est interdit de copier vos images.


<script type="text/javascript" src="https://tscopy.github.io/tscopy.js"></script>
<body onload="tscopy()">



Vous pouvez même asseyer de telecharger l'image si dessou. Bien évidament il est pas impossible de télécharger vos image mais se script repousera plus 90% des voleur d'images!
